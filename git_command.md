## install
`sudo apt-get install git-all`

## config settings and config files
`git config --list --show-origin`

## user info
`git config --global user.name <user-name>`
`git config --global user.email <e-mail>`

## text editor
`git config --global core.editor emacs`

## all configs
`git config --list`

## get help
`git help <verb>`  
`git <verb> help`  
`man git-<verb>`
etc: git help config
